### Please if you have not already have python venv installed
### Use this command 
```
python -m venv .venv
```

### This command to activate the venv
```
.venv\Scripts\Activate.ps1
```

### Download dependincies
```
pip install -r requirements.txt
```

```
typer main.py run --help
```

### Basic search
```
typer main.py run search 'Term'
```

### Deatiled search
```
typer main.py run detailed 'Term
```
